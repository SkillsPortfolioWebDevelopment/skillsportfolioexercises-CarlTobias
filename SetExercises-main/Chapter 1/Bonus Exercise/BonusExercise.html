A Beginner's Guide to HTML and CSS

Basic HTML Tags

Initial Tags

When coding with HTML, one must use two main containers.

The < head > tag is a container for metadata. It has all the information that users can't see, but crucial for the file.

And the < body > tag is where the visible information is stored. Most of the things inside here is seen by the user.

Ex: 

                        
                            < head >
                                < meta charset="UTF-8" >
                                < title >A Beginner's Guide to HTML and CSS< /title >
                                < link rel="stylesheet" href="style.css" >
                            < /head >
                        
                    
                        
                            < body >
                                < p >
                                    Lorem ipsum dolor sit amet, consectetur adipiscing elit.
                                < /p >
                            < /body >

Header Tags

Header tags are used in a variety of degrees. They go from sizes 1 through 6 (1 being the largest).

Inside a source-code editor, its typed as h with the any number from 1-6 beside it and is enclosed with delimiters.

Note: When using tags, it's done in pairs. One opening tag and another closing tag. The closing tag is the same as the opening but with a forward-slash inside the tag.

Ex: 

                        
                            < body >
                                < h1 > Title < /h1 >

                                < h2 > Hello World! < /h2 >
                                < h6 > Lorem ipsum dolor sit amet, consectetur adipiscing elit.< /h6 >
                            < /body >

Paragraph Tag

The paragraph tag is used to represent a paragraph on a webpage. It helps separate text from other blocks adjacent to it.

Inside a source-code editor, its typed as p enclosed with delimiters.

Ex: 

                        
                            < body >
                                < h1 > Hello World! < /h1 >
                                < p > Lorem ipsum dolor sit amet, consectetur adipiscing elit.< /p >
                            < /body >

Bold, Italic, and Underline Tags

In HTML, adding prominence to text is simple. There are three main ways to emphasize text.

There's the bold (< bold >) tag, the emphasis (< em >) tag, and the underline (< u >) tag.

These are used to bolden, italicize, and underline text respectively.

Ex: 

                        
                            < body >
                                < h1 > Hello World! < /h1 >
                                < h6 >
                                    < u >Lorem ipsum< /u > < bold >dolor sit amet < /bold>, 
                                    consectetur adipiscing < em >elit< /em >.
                                < /h6 >
                            < /body >

Div Tag

To semantically divide or separate sections in a webpage, a programmer must use the < div > tag.

In more basic terms, it's a a less prominent < body > tag. It holds the same tags as < body > and creates a section for those tags.

Ex: 

                        
                            < body >
                                < h1 > Title < /h1 >

                                < div >
                                    < h2 > Hello World! < /h2 >
                                    < p > Lorem ipsum dolor sit amet, consectetur adipiscing elit.< /p >
                                < /div >
                            < /body >

Image Tag

When making a webpage, images play a very important role.

In order to insert them, the image (< img >) tag is used. It's also accompanied with src="" for the program to be able to locate the image you want to input.

Ex: 

                        
                            < body >
                                < h1 > Hello World! < /h1 >

                                < div >
                                    < h2 > Image < /h2 >
                                    < img src="img.jpg" >
                                < /div >
                            < /body >

Anchor Tag

Besides images, clickable links are also important.

In order to insert them, the anchor (< a >) tag is used. Unlike the image tag, this one is accompanied with href="" for the program to be able to locate the link you want to input.

Note: The difference between src and href is simple. Using src embeds a resource, while using href identifies the location that an anchor tag may use.

Ex: 

                        
                            < body >
                                < h1 > Hello World! < /h1 >

                                < a href="link.com">Click this link!< /a >

                                < div >
                                    < h2 > Image < /h2 >
                                    < a href="link.com">
                                        < img src="img.jpg" >
                                    < /a >
                                < /div >
                            < /body >

Basic CSS Attributes

font-family, font-size, and font-weight

Using the default font is alright, but to add more pizzaz, one can use the font-family, font-size, and font-weight attributes.

This CSS attribute changes the default font to any font of your liking. There are some presets everyone has, but you can also import fonts from GoogleFonts (which we won't be tackling yet).

Ex: 

                        
                            p {
                                font-family: 'Times New Roman', Times, serif;
                                font-size: 32px;
                                font-weight: bold;
                            }

background-color and color

Text alone is sometimes not enough, sometimes a change in background color is needed in order to give a pop of color.

The main purpose of the background-color attribute is quite straightforward. It changes the background color. It could be specific to text or even the whole body or just the div.

Now, on the other hand, the color attribute changes the color of the font itself instead of its background.

Ex: 

                        
                            body {
                                background-color: cornflowerblue;
                            }

                            div {
                                background-color: rgb(60, 179, 113);
                            }

                            h2 {
                                background-color: #746AB0;
                                color: black;
                            }

height and width

Letting the computer decide the height and width of something is sometimes not very ideal.

The height attribute changes the specified area above and below a given tag, while the width attribute changes the area to the left and right.

Ex: 

                        
                            div {
                                background-color: rgb(60, 179, 113);
                                width: 300px;
                                height: 100px;
                            }

                            h2 {
                                background-color: #746AB0;
                                color: black;
                                width: auto;
                                height: 50px;
                            }

margin and padding

Working with HTML and CSS might not give the wanted spacing between elements you want automatically.

This is where the margin and padding attributes come in.

Each tag has a specific area to it. Now, these areas have borders. Without the padding attribute, any text or image will stay stuck to that border. And without margin, those areas wont have any spacing outside its borders.

To put it more simply, margin give space from the border out and padding help with space from border in.

Note: You can add suffixes like -top, -bottom, -left, and -right to specify space in said direction.

Ex: 

                        
                            div {
                                background-color: rgb(60, 179, 113);
                                width: 300px;
                                height: 100px;

                                margin: 10px;
                                padding: 5px;
                            }

                            h2 {
                                background-color: #746AB0;
                                color: black;
                                width: auto;
                                height: 50px;

                                margin-top: 20px;

                            }
